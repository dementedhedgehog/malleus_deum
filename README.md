# Malleus Deum

Malleus Deum is a fantasy pen and paper, tabletop, role-playing game (RPG).	


# Installation

## Linux

 * Install git.
 * git clone https://github.com/dementedhedgehog/malleus_deum.git
 * Install xelatex.
 * ./publish


## Windows

 * Install Python 2.7+
 * Install MikTek
 * Update MikTek as administrator.
 * Install git.
 * publish.bat
